# App de Canciones

Una aplicación web para gestionar canciones con acordes, transposición y listas de reproducción.

## Características

- Agregar canciones con letras y acordes
- Transponer el tono de las canciones
- Crear y gestionar listas de reproducción

## Cómo usar

1. Ejecuta `npm run dev` para iniciar el servidor de desarrollo.
2. Abre http://localhost:5173 en tu navegador.
3. Agrega canciones usando el formulario.
4. Usa los botones para transponer y agregar a listas.

## Instalación

```bash
npm install
npm run dev
```
